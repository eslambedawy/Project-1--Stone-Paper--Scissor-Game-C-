# 🎮 Stone–Paper–Scissor Game (C++)

A console-based implementation of the classic **Stone–Paper–Scissor** game built using **C++**.  
The player competes against the computer through multiple rounds while the program manages game logic, score tracking, and final results.

This project focuses on practicing **structured programming**, clean function design, and game logic implementation.

# 🕹️ How the Game Works

1. The player selects a choice:
   - `1` → Stone  
   - `2` → Paper  
   - `3` → Scissor  

2. The computer randomly selects its move.

3. The program compares both choices and determines the winner of the round.

4. The game tracks:
   - Player wins
   - Computer wins
   - Draws

5. After all rounds finish, the **final winner** is displayed along with the full game summary.
